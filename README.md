# My Neovim confguration

#### Clone the repository to .config/nvim

```shell
mv ~/.config/nvim ~/.config/nvim.bk
git clone https://github.com/jcolethornton/nvim_config> ~/.config/nvim
```

#### Start Neovim

```shell
nvim
```

#### LazyVim
Install plugins
```shell
leader p
```
